Microsoft Edge Mobile Apps Feedback Analysis Report

Executive Summary

This report presents an analysis of user feedback for Microsoft Edge mobile applications (Android and iOS) collected from May 7, 2025 to May 20, 2025. The analysis covers 977 user reviews from both platforms, with 877 from Android and 100 from iOS.

Key Findings

  • Overall Sentiment: Average rating is 3.72 out of 5.0 (Android: 3.70, iOS: 3.85)
  • Top Category: Performance (247 mentions)
  • Feature Requests: 185 users requested new features or enhancements
  • Bug Reports: 156 users reported bugs or issues

Typical user reviews reflecting overall sentiment:

Android 4 "Microsoft Edge is a solid browser with excellent features. I love the syncing with my desktop and the built-in ad blocking. It's definitely faster than Chrome on my device and uses less battery. Only wish it had better tab organization."
iOS 5 "Feature reach and ease of use for anyone in the MS ecosystem. Love how it syncs with my desktop and the reading mode is excellent."
Android 2 "Edge used to be my favorite mobile browser but since the last update it crashes constantly. Please fix the stability issues!"

Methodology

Data Sources

  • Google Play Store reviews for Microsoft Edge Android (app ID: com.microsoft.emmx)

Analysis Approach

The analysis employed natural language processing techniques to categorize feedback into primary categories and subcategories, determine sentiment scores, and identify feature requests and bug reports. The review text and star ratings were used together to provide a comprehensive understanding of user sentiment.

Limitations

  • Automated categorization may misclassify some feedback due to language nuances.
